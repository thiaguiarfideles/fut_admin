Jogadores Confirmados

Tem como objetivo mostrar aos usuários quais jogadores confirmaram sua participação em partidas específicas. Essa rota é útil para os administradores da liga ou para os próprios jogadores, pois fornece uma maneira rápida e conveniente de verificar quem está disponível para jogar em cada partida agendada. Isso pode ajudar a garantir que as equipes estejam completas e prontas para competir em cada jogo.

Organização de partidas de futebol

Escolha dos times: Essa rota e função são importantes para organizar os jogadores em times de maneira justa e eficiente para as partidas de futebol
Embaralhar a lista de jogadores: A lista de jogadores é embaralhada aleatoriamente para garantir que os times sejam balanceados.
Inicializar os times: Uma lista de listas vazias é criada para representar os times. O número de listas na lista é igual ao número total de jogadores dividido pelo tamanho do time.
Separar os goleiros: Os jogadores que têm "Goleiro 2" em seus nomes são separados como goleiros. Os outros jogadores são armazenados em uma lista separada.
Distribuir os goleiros: Os goleiros são distribuídos nos dois primeiros times.
Distribuir os outros jogadores: Os outros jogadores são distribuídos nos times restantes. Para cada time, a função verifica primeiro se o time já tem três jogadores com menos de 25 anos.
Se não tiver, o jogador é adicionado ao time. Se o time já tiver três jogadores com menos de 25 anos, o jogador é adicionado ao time somente se o time não estiver cheio.
Criar uma lista numerada de times: Uma lista de tuplas é criada, onde cada tupla contém o número do time e a lista de jogadores naquele time.
